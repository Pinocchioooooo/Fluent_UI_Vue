## API

<details>
<summary>button</summary>

#### Attributes

| 参数    | 说明           | 类型    | 可选值 | 默认值 |
| ------- | -------------- | ------- | ------ | ------ |
| type    | 类型           | string  | text   | \-     |
| icon    | 图标类名       | string  | \-     | \-     |
| disable | 是否禁用状态   | boolean | \-     | false  |
| plain   | 是否为朴素按钮 | boolean | \-     | false  |

</details>
